Contains MATLAB and Python code to solve Aiyagari (1994).

Directory guide:
./Python - Code written in Python to solve Aiyagari (1994).
./Python/Main.ipbyn - Main Jupyter Notebook script to solve model.
./Python/Functions - Auxiliary functions used in the main script.
./MATLAB - Code written in MATLAB to solve Aiyagari (1994).
./MATLAB/Main_VFI.m - Main script to solve model using Value Function Iteration with (linear) interpolation and Howard improvement.  
./MATLAB/Main_PFI.m - Main script to solve model using Policy Function Iteration with (linear) interpolation.
./MATLAB/Main_EGM.m - Main script to solve model using Endogenous Grid Method.
./MATLAB/Functions - Auxiliary functions used across main scripts. 
./MATLAB/Figures - Figures relating to solution.
