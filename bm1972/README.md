Contains MATLAB code to solve the Business Cycle model in Brock and Mirman (1972).

### Directory guide: 
./Main_VFI - Main script to solve model using Value Function Iteration with (linear) interpolation and Howard improvement.  
./Main_PFI - Main script to solve model using Policy Function Iteration with (linear) interpolation.  
./Functions - Auxiliary functions used across all main scripts.  
./Figures - Figures relating to solution.  

### References 
Brock, W. A., & Mirman, L. J. (1972). Optimal economic growth and uncertainty: the discounted case. Journal of Economic Theory, 4(3), 479-513.
